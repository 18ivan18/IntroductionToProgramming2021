# FAQ
Как декларирам функция?
```c++
void doSomething() {
    1 + 2;
}
```

Как извиквам функция?
```c++
int sum(int a, int b) {
    return a + b;
}

int a, b;
std::cin >> a >> b;
std::cout << sum(a, b);
```

Трябва ли имената на формалните параметри да са същите като имената на фактическите параметри?
- tl;dr не

```c++
void doSomething(int firstNumber, int secondNumber) {
    // ...
}

// ...
int a = 5, b = 6;
doSomething(a, b);
```
